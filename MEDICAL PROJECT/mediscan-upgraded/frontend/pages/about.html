About MediScan AI

A full-stack AI health assistant built as a BCA final year project, demonstrating real-world application of AI in healthcare.

The Project

What is MediScan AI?

MediScan AI is a full-stack web application developed as a BCA (Bachelor of Computer Applications) Final Year Project. It demonstrates how modern artificial intelligence can be applied to the healthcare domain to improve access to preliminary medical guidance.

The system allows users to describe their symptoms in natural language and receive an intelligent, conversational analysis — including possible conditions, self-care recommendations, and guidance on whether to seek emergency care.

The Problem We Solve

Millions of people face uncertainty when they experience health symptoms. Long waiting times, limited access to specialists, and unreliable internet searches leave patients confused and anxious. MediScan AI provides an instant, preliminary first step — helping people understand their symptoms before visiting a doctor.

How It's Built

The project uses a decoupled architecture: a pure HTML/CSS/JavaScript frontend communicates with a Python Flask REST API backend, which in turn calls the Anthropic Claude AI model. This separation reflects real-world production patterns used by professional development teams.
