The Process

How MediScan AI Works

From the moment you describe a symptom to when you receive your analysis — here's exactly what happens behind the scenes.

Step by Step

Your Consultation Journey

1

🖥️ Open the Consultation Page

Visit the Consultation page on the MediScan AI website. The HTML/JS frontend loads in your browser and checks whether the Python Flask backend is running and reachable on your local machine.

Frontend → Status Check
2

💬 Describe Your Symptoms

Type your symptoms in the chat input in plain, natural language. You can also tap one of the quick symptom buttons to get started immediately. Examples: "I have a headache and feel nauseous" or "My chest feels tight and I'm short of breath."

User Input
3

📡 Frontend Sends Request to Python

JavaScript uses the fetch() API to send a POST request to http://localhost:5000/api/chat — your local Python Flask server. The entire conversation history is sent as a JSON array so the AI has full context.

HTTP POST → Flask Backend
4

🐍 Python Flask Processes the Request

The Flask server receives the conversation, validates the message format, and uses the official Anthropic Python SDK to call the Claude AI model. The medical system prompt is injected server-side — keeping the AI's behaviour controlled and safe.

Flask → Anthropic SDK
5

🤖 Claude AI Analyses Symptoms

The Anthropic Claude model analyses the full conversation context. It follows a carefully crafted clinical system prompt that instructs it to ask follow-up questions, suggest 2–4 possible conditions, provide self-care guidance, and flag emergencies. It never provides a definitive diagnosis.

AI Analysis
6

📬 Response Returns to Your Browser

The AI's reply travels back through Flask → HTTP response → JavaScript → and is displayed as a chat bubble in the browser. The frontend detects emergency keywords and shows a prominent red 911 alert banner when needed. The conversation continues until the user has all the information they need.

Response → Display
